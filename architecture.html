Mail Intelligence Architecture

Outlook 메일 AI 트라이아지 시스템 — 로컬 AI + 하이브리드 폴백

프론트엔드

  • • Vanilla JS (ESM 모듈)
  • • CSS Grid 3칼럼 레이아웃
  • • 드래그 리사이저
  • • 실시간 피드백 UI
  • • 메일 검색/필터링

서버

  • • Node.js HTTP 서버 (ESM)
  • • Microsoft Graph API 통합
  • • OAuth 2.0 + PKCE
  • • JSON 파일 기반 저장
  • • AI 프로바이더 관리

AI 분석

  • • 하이브리드 3단계 폴백
  • • F-AIOS-v3 (로컬 기본)
  • • LM Studio (1차 폴백)
  • • Gemini (2차 폴백)
  • • 분석 결과 캐싱

데이터

  • • mail-cache.json (메일 + 분석)
  • • runtime-config.json (설정)
  • • .outlook-config.json (OAuth)
  • • accounts.json (계정)
  • • attachment-archive.json (첨부)

API 엔드포인트

  • • GET /api/outlook/status
  • • GET /api/outlook/messages
  • • GET /api/outlook/analyze
  • • POST /api/outlook/send
  • • POST /api/outlook/feedback

핵심 기능

  • • 메일 자동 분류 (긴급/진행/대기/완료)
  • • 다음 액션 추천 (3시나리오)
  • • 사용자 피드백 학습
  • • 메일 발송 (Outlook 연동)
  • • 증분 동기화 (캐시 기반)
