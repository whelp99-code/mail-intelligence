## 1. Inbox Zero 스타일 - AI 자동 분류 + 원클릭 처리

### 1.1 목표
- AI가 메일을 자동으로 분류하고, 사용자는 확인만 하면 되는 워크플로우
- 원클릭으로 아카이브/리마인드/회신 처리

### 1.2 구현 항목

| 항목 | 설명 | 난이도 |
|------|------|--------|
| AI 자동 분류 | `enrichWithAI()` 결과를 UI에 바로 반영 | 중 |
| 원클릭 아카이브 | 메일 카드에 아카이브 버튼 추가 | 하 |
| 원클릭 리마인드 | 메일 카드에 리마인더 설정 버튼 추가 | 중 |
| 스마트 규칙 학습 | 피드백 기반 자동 분류 규칙 생성 | 중 |

### 1.3 구현 방안

```javascript
// 스마트 규칙 자동 적용
function applySmartRules(message, feedbackHistory) {
  const senderRules = feedbackHistory.filter(f => f.sender === message.from);
  if (senderRules.length > 2) {
    const mostFrequent = senderRules.sort((a, b) => 
      senderRules.filter(r => r.userStatus === a.userStatus).length -
      senderRules.filter(r => r.userStatus === b.userStatus).length
    )[0];
    return mostFrequent.userStatus;
  }
  return null;
}
```

## 7. 기술 스택 업그레이드 (선택)

### 7.1 프론트엔드

| 현재 | 고도화 | 이유 |
|------|--------|------|
| 바닐라 JS | React/Vue + TypeScript | 컴포넌트 기반 개발, 타입 안전성 |
| CSS only | Tailwind CSS + Headless UI | 일관된 디자인 시스템 |
| localStorage | IndexedDB + Service Worker | 대용량 데이터, 오프라인 지원 |
| 동기 로딩 | React Query + 무한 스크롤 | 성능 최적화 |

| 현재 | 고도화 | 이유 |
|------|--------|------|
